# KubeSphere

## 1. 准备资源

### 1.1 查看 KubeKey 最新版本

~~~shell
curl --silent "https://api.github.com/repos/kubesphere/kubekey/releases/latest" | grep '"tag_name":' | sed -E 's/.*"([^"]+)".*/\1/'
~~~

### 1.1 下载 KubeKey

~~~shell
export KKZONE=cn
curl -sfL https://get-kk.kubesphere.io | VERSION=v3.0.7 sh -
~~~
